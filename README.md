Networking-labs-Project
My Comptia Network+ and CCNA lab projects (VLANs, VoIP, Routing) built using CISCO Packet Tracer
This project demonstrates the design and implementation of a small enterprise network using Cisco Packet Tracer. The network integrates wired and wireless devices, VLAN segmentation and inter-VLAN routing.

The goal of this lab is to simulate a real-world business environment where multiple departments and device types communicate securely and efficiently across the network.

This lab demonstrates:

* VLAN configuration (DATA & VOICE)
* Inter-VLAN routing (Router-on-a-Stick)
* Wireless access via Access Point

Topology

* 2 Switches (2960)
* 1 Router (ISR4321)
* PCs, Laptops, Tablet

Key Configurations

* VLAN 2 → DATA
* VLAN 3 → VOICE
* Trunk links between switches and router


What I Learned

* How to configure voice VLAN
* Inter-VLAN routing setup
* Wireless VLAN challenges
