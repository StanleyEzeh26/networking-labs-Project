# networking-labs-Project
My Comptia Network+ and CCNA lab projects (VLANs, DHCP, VoIP, Routing) built using CISCO Packet Tracer
This project demonstrates the design and implementation of a small enterprise network using Cisco Packet Tracer. The network integrates wired and wireless devices, VLAN segmentation, DHCP services, and inter-VLAN routing.

The goal of this lab is to simulate a real-world business environment where multiple departments and device types communicate securely and efficiently across the network.
